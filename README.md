# 🎓 EduStream Hub - Professor's Management System
**BCA Final Semester Project | Jamuni Lal College, Hajipur**

EduStream Hub ek modern web application hai jo professors ke daily tasks jaise student attendance aur record-keeping ko digital aur asaan banane ke liye design kiya gaya hai.

## 🚀 Key Features
*   **Digital Attendance:** Ek hi dashboard se students ki attendance (Present/Absent) mark karne ki suvidha.
*   **Real-time Analytics:** Total students, Present, aur Absent ka count dashboard par live dikhta hai.
*   **Search & Filter:** Kisi bhi student ko unke naam se turant dhundne ke liye advanced search bar.
*   **CSV Export:** Attendance details ko ek click mein Excel-ready CSV format mein download karein.
*   **Responsive UI:** Mobile aur Desktop dono screens ke liye optimized interface.

## 💻 Tech Stack
*   **Frontend:** React.js (Hooks & Functional Components)
*   **Styling:** CSS3 (Custom Modules & Flexbox)
*   **Language:** JavaScript (ES6+)
*   **Version Control:** Git & GitHub

## 📂 Folder Structure
* `src/App.jsx` - Main application logic aur states.
* `src/App.css` - Custom design aur layout styling.
* `src/assets/` - Project mein use ki gayi images aur icons.
* `package.json` - Project dependencies aur configurations.

## 🛠️ Local Setup
Project ko apne machine par chalane ke liye:
1. Repo clone karein: `git clone [aapka-repo-link]`
2. Folder mein jayein: `cd edustream-hub`
3. Dependencies install karein: `npm install`
4. Project start karein: `npm start`
